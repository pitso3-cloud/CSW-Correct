# CSW Correct - Document Compliance Checker\n\nA desktop application for checking document compliance against CSW (Comprehensive Software Writing) standards.\n\nBuilt with **Electron**, **React**, **TypeScript**, and **Vite**.\n\n## Features\n\n- **Document Checker**: Upload and analyze documents for CSW compliance\n- **Quick Reference Tools**: Search abbreviations and military ranks\n- **Reference Guide**: Browse CSW standards and guidelines\n- **Real-time Compliance Scoring**: Get instant feedback on document quality\n\n## Tech Stack\n\n- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS\n- **Desktop**: Electron 30 with context isolation\n- **Database**: sql.js (WebAssembly SQLite)\n- **Build**: Electron Builder\n\n## Getting Started\n\n### Prerequisites\n\n- Node.js 18+\n- npm or yarn\n\n### Installation\n\n```bash\ngit clone https://github.com/pitso3-cloud/CSW-Correct.git\ncd CSW-Correct\nnpm install\n```\n\n### Development\n\n```bash\n# Start dev server and Electron\nnpm run dev\n```\n\n### Build\n\n```bash\n# Build for production\nnpm run build\n```\n\n## Architecture\n\n### Main Process\n
- `electron/main.ts` - Application entry point, window management, IPC handlers\n- `electron/preload.ts` - Context bridge for secure IPC communication\n- `electron/database.ts` - Database layer using sql.js\n- `electron/engine.ts` - Document validation logic\n- `electron/parser.ts` - Document parsing utilities\n\n### Renderer Process\n
- `src/App.tsx` - Main application router and layout\n- `src/pages/*` - Page components (Home, Checker, Reference, QuickTools)\n- `src/components/*` - Reusable UI components\n- `src/electron-shim.ts` - Browser-safe mock API for development\n\n## IPC Communication\n\nThe app uses secure IPC channels for main-renderer communication:\n
- `check-document` - Analyze document compliance\n- `get-abbreviations` - Query CSW abbreviations\n- `get-ranks` - Fetch military rank information\n\n## Development Notes\n\n- The browser shim (`electron-shim.ts`) provides mock APIs so you can test the UI in a browser without Electron\n- Database is file-based using sql.js; persisted to `~/.csw-correct/csw_database.db`\n- Vite HMR is enabled for fast dev iteration\n\n## License\n
